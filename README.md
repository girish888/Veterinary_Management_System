A modern full-stack Veterinary Management System built with Spring Boot, MySQL, Thymeleaf, and JavaScript. Features include multi-role login (Admin, Vet, Pet Owner), smart appointment scheduling, automated email reminders, real-time notifications, profile photo upload, dashboards for every role, and a powerful admin panel for system management. Designed with secure authentication, responsive UI, fast performance, and complete pet-care workflow automation.
